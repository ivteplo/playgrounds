# Playgrounds
Little coding projects that I make for fun

## Development

### Prerequisites
- Node.js and npm

### Setup

1. Clone the repository
```bash
git clone https://github.com/ivteplo/playgrounds
```

2. Navigate to the folder
```bash
cd playgrounds
```

3. Install dependencies
```bash
npm install
# if you prefer yarn:
yarn install
```

4. Start development server
```bash
npm run dev
```
